🩺 About Our Project

Learn how our AI-powered system helps predict diabetes early using Machine Learning and modern web technologies.

Project Overview

The AI-Based Early Diabetes Prediction System is designed to predict diabetes risk using Machine Learning techniques. Users can enter health information and receive an instant prediction.

Project Objectives

  • Detect diabetes at an early stage.
  • Provide fast and accurate predictions.
  • Help users understand their health risk.
  • Use Artificial Intelligence in healthcare.

Technologies Used

  • Python
  • Django Framework
  • HTML
  • CSS
  • Machine Learning
  • Scikit-Learn
  • SQLite Database

Dataset Information

This project uses the Pima Indians Diabetes Dataset, which contains medical information such as glucose level, BMI, blood pressure, age, insulin, and other health factors.

Machine Learning Model

The system uses a trained Machine Learning model to analyze patient health parameters and predict whether a person is likely to have diabetes.

Future Scope

  • Prediction History
  • PDF Report Generation
  • Email Notifications
  • Cloud Deployment
  • Advanced AI Models
